Welcome to my Page!

This site demonstrates (in-theory) accessible principles

This text should be readable by a screen reader. HTML markup has been designed, especially in HTML5, to better handle use of screen readers. This works as long as proper HTML tags are used, such as Nav or 'h1'. Otherwise the screen reader may get confused!

Feel free to look around. This site is a demo, so there is not much here. I made this all myself so there are only the basics, but I hope they demonstrate effectively some principles of effective and accessible design. View 'articles', which are things I wrote or debated while taking CS 3650. There's some demonstration of accessible design that will show how this class has taught me about designing web pages. There is also an 'about' page if you want to learn about me.
